My Projects

Collaborative Project: Full-Stack Memory Game

A full-stack memory game built with my classmate during the HackYourFuture bootcamp. The game challenges players to match pairs of cards by flipping them over. The frontend uses HTML, CSS and JavaScript, while the backend is built with Node.js and Express.js with card data stored in an SQLite database.

Features

  • Backend & Database — Cards are stored in an SQLite database and served via a backend API. New cards can be added without changing the frontend.
  • Responsive Design — Fully mobile-friendly layout with dynamic card sizes and grid adjustments.
  • Three Difficulty Levels — Each level changes card size, grid gap, and game complexity.
  • Game Mechanics — Click to reveal emojis, match pairs, track time, reset anytime, and finish when all pairs are matched.

Tech Stack

HTML, CSS, JavaScript (ES6+), Node.js, Express.js, SQLite

My Role

I implemented the matching logic, reveal count, timer and card‑disappearance functionality. I also collaborated on styling the game board with CSS.
